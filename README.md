# 🍬 Doces da Nega

Site de pedidos online para a doceria **Doces da Nega**, feito em **HTML, CSS e JavaScript**.  
O site permite que clientes escolham produtos, adicionem ao carrinho e finalizem pedidos via WhatsApp de forma prática e rápida.

---

## 🌟 Funcionalidades

- Carrinho de compras funcional com contador atualizado  
- Modal do carrinho que **abre apenas ao clicar no botão flutuante**  
- Adicionar, remover e limpar produtos do carrinho  
- Finalizar pedido via WhatsApp com mensagem automática  
- Filtro de produtos por categoria  
- Design responsivo e moderno, compatível com desktop e mobile  

---

## 🛠 Tecnologias usadas

- **HTML5** – Estrutura da página  
- **CSS3** – Layout, cores, animações e responsividade  
- **JavaScript (ES6+)** – Funcionalidades do carrinho e interação  
- **LocalStorage** – Persistência dos itens do carrinho  

---

## 🚀 Como usar

1. Abra o site pelo GitHub Pages ou localmente (`index.html`)  
2. Clique nos produtos para adicionar ao carrinho  
3. Clique no **botão flutuante do carrinho** para visualizar ou finalizar o pedido  
4. Ao finalizar, a mensagem é enviada automaticamente via WhatsApp
